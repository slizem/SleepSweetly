Монитор процессов, блокирующих автоматический переход Windows в спящий режим.

## Зачем

Windows не засыпает из-за программ, которые удерживают систему активной. SleepSweetly показывает, какие именно процессы мешают сну, чтобы не лезть в терминал и не искать нужные программы.

## Инструкция

При запуске запросит права администратора. Работает фоново в трее, выполняя запросы только по просьбе пользователя. В корне проекта лежит скрипт, имитирующий процесс удержания ОС от перехода в режим сна для тестирования работоспособности.
